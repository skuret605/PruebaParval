# Prueba Técnica Parval

Este repositorio contiene dos proyectos:

## 🖥️ Frontend (React)
Ubicación: `./frontend`

- SPA que simula un portafolio de inversiones
- TailwindCSS
- Estado con `useState`

## 🔐 Backend (Node.js)
Ubicación: `./backend`

- API RESTful con autenticación JWT
- Control de acceso por roles (`admin` y `analyst`)

## 🚀 Cómo ejecutar

### Frontend
```bash
cd frontend
npm install
npm start
